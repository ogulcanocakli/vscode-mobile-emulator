# VSCode Mobile Emulator

A Visual Studio Code extension for mobile device emulation and development.

## Features

- Mobile device emulation capabilities
- Development tools for mobile applications

## Installation

You can install this extension from the VSCode marketplace or package it manually using `npm run package`.

## Usage

After installation, access the mobile emulator features through the command palette.
